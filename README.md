# 🐾 Simone Martins – Pet Sitter

Site institucional da **Simone Martins – Pet Sitter**, desenvolvido para apresentar serviços de cuidados domiciliares para cães e gatos, com foco em carinho, rotina, segurança e atendimento personalizado.

🔗 **Acesse o site:**  
https://simonemartins-pet.github.io

---

## 📌 Sobre o Projeto

Este projeto tem como objetivo oferecer uma presença digital clara, profissional e acolhedora para tutores que buscam:

- Hospedagem domiciliar
- Banho em casa
- Visitas com cuidados especiais
- Atendimento humanizado e de confiança

O site foi pensado para ser:
- Leve e rápido
- Fácil de manter
- Responsivo (desktop e mobile)
- Escalável para novas páginas e funcionalidades

---

## 🛠️ Tecnologias Utilizadas

- **HTML5** – Estrutura das páginas
- **CSS3** – Estilos personalizados
- **Tailwind CSS (CDN)** – Layout moderno e responsivo
- **JavaScript (Vanilla)** – Interações dinâmicas
- **Font Awesome** – Ícones
- **Google Fonts** – Tipografia
- **GitHub Pages** – Hospedagem
